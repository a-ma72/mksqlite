Willkommen beim Projekt mksqlite!

Wollten Sie jemals Ihre MATLAB Daten in einer SQL Datenbank speichern?
Nichts großes, einfach nur ein paar Arrays oder Strukturen in ein paar Tabellen.
Vielleicht ein paar Indicies und das ganze über SQL Befehle verwaltbar?

Dann ist mksqlite das Richtige für Sie!

mksqlite verbindet die Power von MATLAB mit der Leistungsfähigkeit der
Datenbankengine SQLite!

Besuchen Sie einfach die Projektseite https://sourceforge.net/projects/mksqlite/files/
oder lesen Sie die mksqlite Dokumentation.

Haben Sie Probleme bei der Verwendung von mksqlite? Dann schauen Sie
doch einmal in die kleine FAQ Liste am Ende der Dokumentation.

Welcome to the project mksqlite!

Did you ever want to store your MATLAB data in a SQL database?
Nothing large, simply a few array or strutures in a few tables.
Maybe a few indicies and everything manageable with SQL commands?

Then msqlite will be the right choice for you!

mksqlite connects the power of MATLAB with the efficiency of the
SQLite database engine.

Just visit the mksqlite projectpage at https://sourceforge.net/projects/mksqlite/files/
or read the Documentation of mksqlite.
